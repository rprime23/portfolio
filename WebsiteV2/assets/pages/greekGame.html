Greek Game

Greek Game is an alternative style game that attempts to help teach the player the reality that Zeus is actually a bad guy. Taking the player through and adventure as one of Zeus' warriors who turns against him, the player encounters other caharcters that attempt to teach them about who Zeus really is.

Greek Game was created in an eight person team. This project was built in Gamemaker using Gamemaker's programming language. I worked as primarily an engineer, but was also a level designer and artist.
